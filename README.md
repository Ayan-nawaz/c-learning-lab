# C Learning Lab

A collection of beginner-friendly C programming programs designed to strengthen core programming concepts. This repository includes examples covering arrays, pointers, memory, loops, conditional statements, and basic problem-solving techniques.

## Programs Included

- `printreverse.c` – Print array elements in reverse order.
- `markslessthan65.c` – Display marks that are less than 65.
- `address.c` – Demonstrates memory addresses using pointers.
- `garbagevalue.c` – Explains uninitialized variables and garbage values.
- `sumofarray.c` – Calculate the sum of all elements in an array.

## Topics Covered

- Arrays
- Loops
- Conditional Statements
- Pointers
- Memory Addresses
- Garbage Values
- Basic Array Operations

## Requirements

- GCC Compiler
- Visual Studio Code / Code::Blocks / Turbo C (optional)

## How to Run

```bash
gcc filename.c -o output
./output
```

Replace `filename.c` with the desired program file.

## Purpose

This repository is created for learning, practicing, and understanding the fundamentals of C programming through simple and easy-to-understand examples.
